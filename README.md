# cunner

A cloud computing platform, where you can test the code output safely.


## Dependencies

|                                       | recommended version |
| ------------------------------------- | ------------------- |
| [Node.js](https://nodejs.org/en)      | 20.16               |
| [MongoDB](https://www.mongodb.com/)   | 7.0.12              |
| [DockerAPI](https://www.docker.com/)  | 1.46                |
| [Redis](https://redis.io/)            | 7.0.8               |
